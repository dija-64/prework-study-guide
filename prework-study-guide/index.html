Git

Useful Git Prompts:

  • pwd - displays the current working directory
  • ls - displays list of files and folders int he current directory
  • cd - navigates between directories in the path
  • cd.. - navigaes backwards one level at a time through the path of directories
  • git status - checks which branch you're currenty in
  • git pull origin main - pulls down the latest main branch
  • git checkout -b branch-name - creates a new branch with new name and switches to it
  • git add -A - commits all files in the working branch
  • git commit -m - associates massage with commit
