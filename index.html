HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The 'p' element represents paragraph of text
  • The 'img' element embeds an image
  • The 'form' element represents a form

CSS

  • Margins: Indicates how mych space wanted around outside of an element. Not visible on page
  • Padding: Adds space around the content inside element. Increase makes entire element bigger
  • * Selector Applies rule to all elements visible on page.

Git

Useful Git Prompts:

  • pwd - displays the current working directory
  • ls - displays list of files and folders int he current directory
  • cd - navigates between directories in the path
  • cd.. - navigaes backwards one level at a time through the path of directories
  • git status - checks which branch you're currenty in
  • git pull origin main - pulls down the latest main branch
  • git checkout -b branch-name - creates a new branch with new name and switches to it

JavaScript

  • Variable:Named continer that allows us to storee data in our code
  • Control Flow:Order in which a computer executes code in a script
  • Console.log:Used to output a message to the web console by adding an argument
  • Array:Single variable used to group related data
  • Index:Unique number identifier which shows location of the dta item in the array
  • Functions:Grouped set of instructions that tell computer how to perform certain task
